# Socket Flow

Чат с WebSocket. Фронт (Vite + React) и бэкенд (Go + Gin).

## Запуск для разработки

Фронт проксирует запросы на бэкенд по адресу `http://localhost:8080`. **Сначала должен быть запущен бэкенд**, иначе в консоли будут ошибки `ws proxy error: ECONNREFUSED` и чат не заработает.

### 1. База данных (Postgres)

```bash
docker compose up -d postgres
```

### 2. Бэкенд (порт 8080)

В корне проекта:

```bash
go run ./cmd
```

В логах должно появиться: `http server started port=8080`.

### 3. Фронт

В отдельном терминале:

```bash
npm run dev --prefix frontend
```

Откройте в браузере адрес, который выведет Vite (например `http://localhost:5173` или `http://localhost:5174`).

---

Если бэкенд не запущен, фронт будет открываться, но вход, регистрация и отправка сообщений не будут работать, а в терминале Vite появятся ошибки прокси WebSocket.
